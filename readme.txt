Shows notifications in wordpress

== Description ==

This plugin is designed to add a notification bar with rotating notifications whereever the user wants.

== Installation ==

Installing "LSP-Notification-Bar" can be done either by searching for "LSP-Notification-Bar" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via GitHub
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
